URLslab icon reference

This preview shows icons from generated sprite in conjunction with external SVG.



You can show icon either via class or directly calling icon sprite in PHP code.



Class method

Add class icn-ICON_NAME to the element where you want to show the icon. It will be placed INSIDE that element as a first child, similar to :before in CSS.

Usage: 

    
        <div class="icn-search">
            … other elements
        </div>

Result: 

    
        <div class="icn-search">
            <svg class="icon icon-search">
                <use xlink:href="PATH_TO/icons.svg#search"></use>
            </svg>
            
            … other elements
        </div>

Or add class icn-after-ICON_NAME to element where you want to show the icon. It will be placed INSIDE that element as a last child, similar to :after in CSS.

Usage: 

    
        <div class="icn-after-search">
            … other elements
        </div>

Result: 

    
        <div class="icn-after-search">
            … other elements
            
            <svg class="icon icon-search">
                <use xlink:href="PATH_TO/icons.svg#search"></use>
            </svg>
            
        </div>

PHP Snippet to use to include icon

    
        <svg class="icon icon-ICON_NAME">
            <use xlink:href="get_template_directory_uri() . '/assets/images/icons.svg#ICON_NAME"></use>
        </svg>
